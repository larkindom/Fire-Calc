# 🔥 FIRE Calculator

An interactive tool that maps your path across every flavor of **Financial Independence / Retire Early** — Coast, Lean, regular, and Fat FIRE — based on your savings rate, portfolio allocation, and return assumptions.

**[▶ Live demo](https://larkindom.github.io/Fire-Calc/)** ·  Built as a single, dependency-free HTML file.

## Features
- **Four FIRE milestones** plotted together: Coast (a rising curve), Lean, regular (4% rule), and Fat FIRE — each with a plain-language definition in the legend.
- **Portfolio allocation** across stocks / bonds / cash, with per-asset real-return assumptions that blend into a single weighted return.
- **Live projection chart** with a hover crosshair showing your exact balance, age, and % of FIRE at any point on the timeline.
- All figures in **today's (real) dollars** — no inflation guesswork.
- Fully client-side: no backend, no tracking, no build step. Open the file and it runs.

## Tech
Plain HTML, CSS, and vanilla JavaScript. The chart is hand-drawn on an HTML `<canvas>` — zero libraries or dependencies.

## Disclaimer
Educational model only. It assumes a constant real return and steady contributions, and ignores taxes, market volatility, and sequence-of-returns risk. Not financial advice.
